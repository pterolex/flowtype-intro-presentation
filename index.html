Flow

  • Статический анализатор типов
  • Поддерживает современный стандарт ECMAScript 2017
  • Разработка Facebook, написан на языке OCaml
  • Windows версия появилась в середине 2016 г. (только х64)
  • Из коробки содержит декларации типов для стандартной библиотеки JavaScript, DOM API и стандартной библиотеки Node.js

Установка и настройка

  • npm install flow-bin --save-dev
  • создание .flowconfig

Синтаксис

  • Расширение языка JavaScript (надмножество)
  • Не поддерживается браузерами

Как подготовить типизированный код для запуска в браузере.

  • Comment syntax – не требуется компиляция
  • Babel: transform-flow-strip-types plugin
  • flow-remove-types

Интеграция с существующими инструментами

  • ESLint: eslint-plugin-flowtype-errors
  • Webpack: remove-flow-types-loader
  • Node.js: flow-node / require hook
